OMV-Extras.org designs value added plugins for [[https://www.openmediavault.org/|Openmediavault]], greatly extending server functionality.  Further, OMV-Extras.org facilitates the installation of Openmediavault on Raspberry PI's, Armbian supported Single Board Computers and 32bit - x86 platforms.

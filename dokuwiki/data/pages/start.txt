OMV-Extras.org designs value added plugins for [[https://www.openmediavault.org/|Openmediavault]], greatly extending server functionality.  Further, OMV-Extras.org facilitates the installation of Openmediavault on Raspberry PI's, Armbian supported Single Board Computers and 32bit - x86 platforms.

The document index contains guides for scripted installs, on the titled platforms.  These documents are detailed and are suitable for beginner to intermediate users.

{{ :divider.png?600 |}}

<HTML><Center><strong>The Quick Index for Experienced Linux Users and Admins</strong></Center></HTML>
\\
\\
  * For a quick installation of omv-extras, on Openmediavault v5, run the following command as root:

wget -O - https://github.com/OpenMediaVault-Plugin-Developers/packages/raw/master/install | bash 

  * Or read the [[https://forum.openmediavault.org/index.php/Thread/5549-OMV-Extras-org-Plugin/|guide]].

  * To install Openmediavault AND omv-extras, follow this [[https://github.com/OpenMediaVault-Plugin-Developers/installScript/|guide]].
